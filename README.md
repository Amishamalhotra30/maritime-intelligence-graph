# Maritime Intelligence Graph Platform

## Overview

Maritime Intelligence Graph Platform is a knowledge graph-based analytics system that transforms unstructured maritime market emails into actionable business intelligence.

The platform extracts:

- Cargo Opportunities (VC)
- Time Charter Opportunities (TC)
- Open Vessel Positions (Tonnage)

and converts them into a maritime relationship graph for analytics and opportunity discovery.

---

## Features

### Email Classification

Automatically classifies maritime emails into:

- VC (Voyage Cargo)
- TC (Time Charter)
- Tonnage

### Information Extraction

Extracts:

- Cargoes
- Vessels
- Brokers
- Ports
- Laycans
- Open Dates

### Knowledge Graph

Builds relationships between:

- Cargoes
- Vessels
- Ports
- Brokers

### Analytics

- Trade Lane Analysis
- Broker Intelligence
- Opportunity Scoring
- Market Snapshot

### Interactive Dashboard

- KPI Cards
- Pipeline Summary
- Cargo Table
- Vessel Table
- Opportunity Dashboard
- Graph Explorer
- Analytics Page
